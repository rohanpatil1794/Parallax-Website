# Parallax Website

A responsive 3D parallax website built with HTML, CSS, and JavaScript using the GSAP animation library.

## Features

- **3D Parallax Effect** — mouse movement drives multi-layer depth using `perspective` and `translateZ`
- **Layered Scene** — 10 mountain layers and 7 fog layers composited for realistic depth
- **Entrance Animation** — layers fade in with a staggered GSAP timeline on page load
- **Responsive Header** — navigation bar with logo, links, search icon, and hamburger menu
- **Vignette Overlay** — radial gradient adds cinematic framing

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- [GSAP 3.10.4](https://greensock.com/gsap/)

## Getting Started

No build step required. Just open `index.html` in a browser, or serve it locally:

```bash
npx serve .
```

Then visit `http://localhost:3000`.
